# MyTasks 📝

**MyTasks** est une application mobile Android développée en **Java**, conçue pour simplifier la gestion des tâches quotidiennes. Elle permet aux utilisateurs de créer, catégoriser et suivre l'avancement de leurs objectifs avec une interface intuitive.

## 🚀 Fonctionnalités
- **Gestion des tâches :** Création facile de tâches avec titre, description et catégorie.
- **Organisation :** Catégorisation personnalisée (Personnel, Études, etc.).
- **Suivi temporel :** Définition de deadlines (date limite) pour une meilleure gestion du temps.
- **Système de priorité :** Attribution d'un niveau de priorité (système d'étoiles) à chaque tâche.
- **Statistiques en temps réel :** Résumé automatique du total des tâches et du nombre de tâches terminées.
- **Persistance des données :** Utilisation de la base de données **Realm** pour un stockage efficace et rapide.

## 🛠 Stack Technique
- **Langage :** Java
- **Base de données :** Realm (NoSQL)
- **Architecture :** Utilisation d'activités, de fragments, de listes dynamiques et de menus contextuels.
- **Environnement :** Android Studio

## 💡 À propos
Ce projet a été réalisé dans le cadre d'un développement complet d'application Android, mettant en pratique les concepts de persistance de données (Realm), la manipulation de fragments, ainsi que la gestion d'interface utilisateur dynamique.
